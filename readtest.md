# Javlibrary
-----------------------------------
Javlibrary爬虫 for Python3，该项目可以实现以下功能:
* 获取最高评分/最受期待榜的作品
* 获取所有演员名及其作品目录地址
* 获取指定演员所有作品
* 获取指定番号磁力链接
* 输出内容到excel

## 安装一下
---------------------------------
### 需要安装以下包
  ```Python
  pip install requests
  pip install BeautifulSoup
  pip install pandas
  pip install selenium
  pip install pymongo
  pip install retrying
  ```
